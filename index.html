Startup.Push

An Ecommerce SaaS App

Startup.Push is a Ruby on Rails web application that I created for practice. The app is a social networking community set up to allow two tiers of membership plans: free plans and paid plans. Once a user chooses a plan, they are able to create a profile and to view other profiles in the community. Paid plans have access to member contact information, while free plans do not.

The app has a relational database for storing user information and profile information. The ecommerce transaction for monthly paid plans is handled through Stripe and no sensitive payment information is stored directly in the database. The Devise gem was utilized for user authentication and login. I used Sendgrid for handling mail and Paperclip for attaching profile pictures. Because the app is hosted on Heroku’s free service, I used the AWS S3 cloud for storing profile pictures. Bootstrap is the front end framework.

A SaaS Bookmark App

As a final project for The New York Code + Design Academy Back End 101 course in New York, I created a Ruby on Rails web application for storing bookmarks titled "markit". Each user has their own login account and the ability to save urls and information about the links that they would like to store. All user and bookmark information can be created, edited and deleted.

The Postgres relational database stores both a user’s information and their bookmarks, though because of a one-to-many relationship, users can only view their own bookmarks. I used Devise for user accounts and Paperclip for user account images. I configured my mailer through plain authentication of a Gmail account, however that is not ideal due to sensitive password information that is vulnerable in the code. Navigation, forms, flash messaging and layout were designed using Bootstrap’s front end framework.

Roscoe's Bakery

A Static Business Website

This responsive static website was a project created for The New York Code + Design Academy's Web Development 100 course that I took in Philadelphia. I created this website from scratch with HTML, CSS and JavaScript. I used a real bakery in California as the copy for this fictitious business and purchased a royalty free photo that serves as both the landing page and the color scheme of the site.

I used media queries, CSS percentage values, em units, floats and the FitText jQuery plugin on the landing page to achieve a responsive design. In the image gallery, I used a jQuery lightbox plugin called Magnific Popup to zoom in on and scroll through photos. I also used Google Maps API to add the map and location of the business on the About page. I promoted the business's social media and most used contact information by highlighting them in the footer.
